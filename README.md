# Student Record Management System 🎓

A console-based C program designed to store and manage student records using file handling. Created during my first year at BNCOE.

### 🛠️ Key Features
* **Add Records:** Save new student details permanently to a file.
* **View Records:** Display all stored information in a formatted table.
* **Search/Update/Delete:** Full management of student data.

### 💻 Tech Stack
* **Language:** C (Programming Language)
* **Concepts:** Structures, File Handling, Input Validation
